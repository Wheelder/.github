# Wheelder

**AI-Powered Research Platform** — A Student Project Since 2020, Innovating Since 2023

Wheelder is a research platform that combines AI-powered search, deep research, and learning tools into a unified experience. Built with PHP, Python, and modern web technologies. Still evolving — focused on research and learning, not yet commercialized.

## Key Innovations

- **Circular Search** — Depth-based research system (levels 1-7) that progressively deepens topic understanding
- **Deep Research / Deep Think** — AI-driven multi-turn research conversations with context retention
- **Digging Deeper** — Progressive topic exploration, level by level
- **Text-to-Speech** — Neural voice synthesis for hands-free research consumption
- **Dual-Panel Layout** — Side-by-side answer + visual content display (since 2023)
- **Research Threads** — Persistent research conversations across sessions
- **Lessons Generator** — AI-powered educational content creation
